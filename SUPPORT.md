# Support

## Product support

Use GitHub Issues in this repository for:

- bug reports
- installation problems
- feature requests
- questions about documented behavior

## Security issues

Do not use public issues for vulnerability reports.

Follow `SECURITY.md` for the security disclosure path.

## Scope

Support is best-effort during the initial public release phase and covers the
current taudit CLI, VS Code extension, Azure DevOps extension, and documented
CI platform support described in `README.md`.
